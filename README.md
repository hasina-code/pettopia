# PetTopia Server

## Purpose

This is the backend server for the PetTopia platform. It provides secure REST APIs for pet management, adoption requests, authentication, authorization, and database operations.

## Live URL

🔗 https://pet-topia-sarver.vercel.app

## Features

- RESTful API for pet management.
- Secure JWT authentication and authorization.
- MongoDB database integration.
- Adoption request approval and rejection system.
- Protected routes using middleware.
- Environment variable configuration for sensitive credentials.

## NPM Packages Used

- Express.js
- MongoDB
- JSON Web Token (JWT)
- Cookie Parser
- CORS
- Dotenv
- Better Auth
